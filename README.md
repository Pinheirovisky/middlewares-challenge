# middlewares-challenge

Aplicação desenvolvida em NodeJs. Objetivo é se aprofundar mais em middlewares

## Rodando a Aplicação

Quer rodar essa aplicação na sua máquina? Siga os comandos abaixo:<br />

Primeiro é necessário clonar o projeto:

```
 git clone https://github.com/Pinheirovisky/middlewares-challenge.git
```

Logo após, instale as dependências através do [Yarn](https://yarnpkg.com/):

```
yarn
```

Por fim, inicie o projeto:

```
yarn dev
```
